# AI自动化实践项目

这个项目用于记录和实践各种AI自动化工具的具体应用案例。

## 项目结构

### Koishi实践/
- 具体的机器人项目
- 实际配置和代码
- 项目日志和问题解决
- 成功案例

### n8n工作流/
- 实际的工作流配置
- 具体场景应用
- 问题排查记录
- 效果评估

### Dify应用/
- LLM应用开发记录
- 具体项目实现
- 调试和优化过程
- 用户反馈
