graph TD
    A[分类数据分析] --> B{处理多少个分类变量?}
    B -- 一个 --> C[拟合优度检验]
    B -- 两个或以上 --> D[列联分析]

    C --> C1[目的: 检验单个分类变量的观测频数是否与某理论比例或期望频数吻合]
    C1 --> C2[原假设H₀: 总体中各类所占比例等于已知概率 p₁, p₂, ..., pᵣ]
    C2 --> C3[检验统计量: Pearson χ²]
    C3 --> C4["χ² = Σ [ (观测数 - 期望数)² / 期望数 ]"]
    C4 --> C5[统计量分布: 近似服从 χ² 分布]
    C5 --> C6[自由度: r - 1 (r为分类数)]
    C6 --> C7[判断标准: 比较计算出的χ²值与临界值或P值]
    C7 --> C8[应用实例: 孟德尔豌豆实验数据验证]

    D --> D1[目的: 考察两个或多个分类变量之间是否相互独立]
    D1 --> D2[数据形式: r x c 列联表]
    D2 --> D3[原假设H₀: 两个变量相互独立]
    D3 --> D4[检验统计量: Pearson χ²]
    D4 --> D5["χ² = Σ Σ [ (观测频数 - 期望频数)² / 期望频数 ]"]
    D5 --> D6["期望频数 = (该行合计数 * 该列合计数) / 总合计数"]
    D6 --> D7[统计量分布: 近似服从 χ² 分布]
    D7 --> D8[自由度: (r - 1) * (c - 1) (r为行数, c为列数)]
    D8 --> D9[判断标准: 比较计算出的χ²值与临界值或P值]
    D9 --> D10[应用实例: 儿童智力发展与营养关系的研究]
