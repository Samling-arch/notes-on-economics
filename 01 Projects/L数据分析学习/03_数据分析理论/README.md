# 数据分析理论

这个文件夹包含数据分析的理论基础、方法论和统计学知识。

## 学习目标
- 掌握数据分析的基本流程和方法论
- 学习描述统计和推论统计的基础知识
- 理解实验设计和A/B测试的原理
- 培养数据思维和批判性思考能力

## 课程计划
1. **统计学基础**
   - 描述统计（均值、中位数、方差等）
   - 概率分布（正态分布、二项分布等）
   - 抽样理论
   - 假设检验
   - 置信区间
   - 相关性与因果关系

2. **数据分析方法论**
   - 数据分析流程（CRISP-DM模型）
   - 问题定义和分析框架
   - 数据收集和准备
   - 探索性数据分析(EDA)
   - 数据解释和结果呈现
   - 数据伦理和隐私

3. **实验设计**
   - A/B测试基础
   - 实验组和对照组
   - 样本量计算
   - 统计显著性
   - 多变量测试
   - 实验偏差和控制
