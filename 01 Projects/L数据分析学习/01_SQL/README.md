# SQL 学习

这个文件夹包含所有与SQL相关的学习资料、课程笔记和项目实践。

## 学习目标
- 掌握SQL基础语法和查询
- 学习数据库设计和管理
- 掌握高级SQL技术和性能优化
- 完成实际SQL项目

## 课程计划
1. **SQL基础**
   - SQL语法基础
   - SELECT查询
   - WHERE条件过滤
   - GROUP BY分组
   - JOIN表连接
   - 子查询

2. **SQL进阶**
   - 窗口函数
   - 存储过程和函数
   - 触发器
   - 事务管理
   - 索引优化
   - 查询性能优化

3. **数据库设计**
   - 数据模型设计
   - 规范化和反规范化
   - 索引设计
   - 数据库安全
